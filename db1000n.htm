Death by 1000 needles (db1000n)
      This was one of the first tools to attack
      Russian propaganda and has since
      remained a best choice for specialist
      software with automated features and
      effective results ever since. Written in
      the GO language, it is able to download
      and update its target list (config)
      automaticly, and distribute an abstract
      data payload recursively for each target.
